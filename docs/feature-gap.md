# 能力差距与实现计划

对比对象：[CookSleep/gpt_image_playground](https://github.com/CookSleep/gpt_image_playground)（下称 GIP）。

GIP 是成熟的 gpt-image-2 全功能客户端，强在「多服务商接入 / Agent 对话 / 本地数据管理 / 部署」；
本项目（Image Studio）强在 GIP 没有的「内置提示词库」（submodule 同步 + 本地缩略图 + 分类）。

本文档记录功能差距，并逐项实现。状态：`[ ]` 待办 / `[x]` 已完成。
